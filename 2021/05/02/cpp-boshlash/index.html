# C++ dasturlash tili

Bu dasurlash tili C tilida kelib chiqqan. G++ kompilyator o'rnatamiz. main.cpp fayl yaratamiz

#include<iostream>
using namespace std;

int main(){
	cout<<"Hello world"<<endl;
	return 0;
}

terminal(consol)ga yozamiz g++ main.cpp (yoki g++ main.cpp -o main) teramiz chiquvchi fayl a.out (yoki main) bo'ladi. Dasturni ishga tushirish uchun ./a.out (yoki ./main) terib ishga tushiramiz Hello world ekranga chiqadi.
